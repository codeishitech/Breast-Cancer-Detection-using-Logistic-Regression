# Breast Cancer Detection using Logistic Regression

This project implements a machine learning model to classify breast tumors as **malignant or benign** using **Logistic Regression**.

The model is trained using the Breast Cancer dataset and evaluated using multiple metrics including accuracy, confusion matrix, and cross-validation.

---

## Problem Statement

Early detection of breast cancer significantly improves treatment outcomes.  
This project demonstrates how machine learning can assist in predicting whether a tumor is malignant or benign based on diagnostic features.

---

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

---

## Machine Learning Pipeline

1. Import dataset
2. Data preprocessing
3. Train-test split
4. Feature scaling
5. Model training using Logistic Regression
6. Model evaluation using:
   - Confusion Matrix
   - Accuracy
   - Cross Validation
   - ROC-AUC score

---

## Model

Algorithm used:

**Logistic Regression**

Why Logistic Regression?

- Interpretable
- Efficient for binary classification
- Common baseline model in healthcare ML tasks

---

## Evaluation Metrics

The model was evaluated using:

- Accuracy Score
- Confusion Matrix
- Precision
- Recall
- F1 Score
- 10-Fold Cross Validation

---

## Results

The model achieved:

- High classification accuracy
- Consistent results across cross-validation folds

This shows that Logistic Regression is a strong baseline for this classification problem.
